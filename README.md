# Guia: Cambiar nombre y contrasena de tu WiFi (Herramienta 10.0.0.1)

Este sitio es una **guia paso a paso en espanol** para ayudar a clientes a **cambiar el nombre de la red (SSID)** y/o la **clave WiFi** usando la **Herramienta de administracion** accesible en `http://10.0.0.1`.

> **Nota:** Los nombres de menus pueden variar segun el modelo de gateway o la version de firmware.

## ? Que incluye
- Instrucciones claras y resumidas (paso a paso)
- Boton para copiar `10.0.0.1`
- Consejos de seguridad para la contrasena
- Seccion de **Solucion de problemas**
- **Checklist** para marcar lo realizado

## ?? Uso rapido
1. Conecta tu equipo a tu red (WiFi o Ethernet).
2. Abre `http://10.0.0.1` en el navegador.
3. Inicia sesion con tus credenciales de administrador.  
   - Por defecto (si nunca se cambiaron): usuario `admin`, clave `password`.
4. Ve a **Gateway ? Connection ? WiFi** (o menu similar).
5. Edita el **SSID** y la **contrasena**. Guarda los cambios.
6. Reconecta tus dispositivos usando la nueva clave.

## ?? Recomendaciones de seguridad
- Contrasena recomendada: **>= 12 caracteres** con mayusculas, minusculas, numeros y simbolo.
- Evita datos personales evidentes en el SSID.
- Si olvidaste la clave de administrador, puede requerirse **reset de fabrica** (mantener presionado `reset` 20–30 s).
